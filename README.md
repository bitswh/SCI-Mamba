## 🛠️环境配置

创建虚拟环境
```bash
conda create -n scimamba python=3.8
```

激活虚拟环境
```bash
conda activate scimamba
```

配置环境
```bash
conda install pytorch==1.13.1 torchvision==0.14.1 torchaudio==0.13.1 pytorch-cuda=11.7 -c pytorch -c nvidia
```
```bash
pip install -r requirements.txt
```
